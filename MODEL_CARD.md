# Model Card: Vietnamese Advertisement Generator (LoRA)

## Model Details

- **Model Name**: `vmhdaica/advertisement-lora`
- **Model Type**: Causal Language Model (LoRA Adapter)
- **Base Model**: [Qwen/Qwen3-0.6B](https://huggingface.co/Qwen/Qwen3-0.6B)
- **Language**: Vietnamese (vi)
- **Task**: Conditional text generation — product advertisement creation
- **Framework**: Hugging Face Transformers + PEFT
- **License**: MIT

## Intended Use

### Primary Use
Generate Vietnamese advertising copy for e-commerce products, given a product name and description as input.

### Intended Users
- E-commerce sellers needing bulk advertisement creation
- Marketing teams seeking draft ad copy for Vietnamese markets
- Researchers studying Vietnamese NLG

### Out-of-Scope Use
- **NOT** a general-purpose chatbot or instruction-following model
- **NOT** suitable for generating factual/medical/legal content
- **NOT** designed for languages other than Vietnamese
- Should **NOT** be used to generate misleading or deceptive advertisements

## Training Data

- **Source**: Vietnamese e-commerce product data scraped from Tiki platform
- **Ad Generation**: Advertisements generated by GPT-4o mini API with structured problem/solution prompts
- **Size**: 80,125 training / 4,452 validation / 4,452 test samples (90:5:5 split)
- **Content**: Product names, descriptions, and corresponding advertisement texts (~650–750 words each)
- **Preprocessing**: Deduplicated by product ID; removed HTML tags and hashtags; advertisements filtered to ≤1,024 tokens; formatted as ChatML conversations

## Training Procedure

- **Method**: LoRA (Low-Rank Adaptation)
- **LoRA Configuration**: r=8, α=16, dropout=0.05, targeting all attention + MLP projections
- **Trainable Parameters**: 5,046,272 / 601,925,632 (0.84%)
- **Hardware**: 2× NVIDIA T4 16GB (Kaggle)
- **Precision**: FP16
- **Optimizer**: AdamW (fused), manual LR decay (3e-4 → 5e-5)
- **Effective Batch Size**: 64
- **Training Strategy**: Epoch-by-epoch with validation monitoring; LR reduced on plateau; best checkpoint selected

## Evaluation

### Automated Metrics (Full Test Set — 4,452 samples)
- **BLEU**: 0.2711
- **ROUGE-1**: 0.8504 (highest among all compared models)
- **ROUGE-2**: 0.5978 (highest among all compared models)
- **ROUGE-L**: 0.4254

Compared against BARTpho-word-base, mT5-Small, and ViT5-base. Qwen3-0.6B achieved the best ROUGE-1 and ROUGE-2 scores.

### Human Evaluation (100-sample subset, scored 0–1)
- **Grammar & Logic**: 0.643
- **Readability**: 0.635
- **Relevance**: 0.688

### Limitations
1. **Domain-specific**: Only trained on e-commerce product ads; may produce irrelevant content for other domains
2. **Style bias**: Outputs reflect the style distribution of the training data (emoji-heavy, promotional tone)
3. **Hallucination risk**: May generate product claims not present in the input description
4. **Length control**: No explicit mechanism to control output length
5. **Vietnamese only**: Not designed for multilingual use
6. **Creativity**: Generated ads may lack brand-specific nuances and stylistic diversity

## Ethical Considerations

- **Deceptive advertising**: Generated ads should be reviewed by humans before publication to ensure compliance with advertising regulations
- **Bias**: The model may reproduce biases present in the training data regarding product categories, pricing language, or cultural stereotypes
- **Synthetic training data**: Advertisements were generated by GPT-4o mini, which may introduce its own biases into the training distribution
